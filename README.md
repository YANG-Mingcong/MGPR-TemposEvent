# MGPR-TemposEvent
Il s'agit d'un projet visant à explorer la performance interactive. Cette application est le middleware de backend du projet, et est utilisée pour recevoir des données de backend du web et les "traduire" en commandes pour les appareils audio et vidéo.

This is a project to explore interactive performance. This application is the backend middleware for this project, and is used to receive backend data from the web and "translate" it into commands for audio and video devices.
